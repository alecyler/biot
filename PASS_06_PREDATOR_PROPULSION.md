# Pass 06 – Predator / Propulsion / Scavenger Push

## Main goals
- Make lightning arcs linger and fade more gently.
- Make propulsion matter more for hunters.
- Give brain/perception builds more control over thrust direction.
- Seed the world with more repeated starter archetypes so fragile predator lineages have backup populations.
- Seed starter carrion / mycelium patches so scavengers and hunters have something to contest immediately.
- Address the "nothing is eating carrion" problem by adding carrion-seeking behavior and broadening scavenger eligibility.

## Mechanical changes
- Propulsion force increased and propulsion energy cost reduced.
- Brain + perception + propulsion now add controlled thrust in the steering direction.
- Hunter drive scales a bit harder with offensive organs and excess energy.
- Hunters now seek prey first, then carrion / mycelium if prey is not nearby.
- Non-photosynthetic non-hunters can also weakly drift toward carrion.
- Carrion pickup range increased.
- Carrion value increased.
- More organ types can consume carrion, including propulsion builds.

## Seeding changes
- Starter seeding now guarantees multiple copies of each main archetype before filling leftovers.
- Random starters still exist, but they are a minority.
- The map now starts with carrion / spore / mycelium patches already present.

## Propulsion identity
- Propulsion is still perpendicular thrust.
- Uncontrolled propulsion still pulses rhythmically.
- Controlled propulsion is approximated by adding thrust in the steering direction for mature brain+perception builds.
- Starter hunter/skirmisher archetypes now use shorter paired propulsion limbs more often.

## Visual changes
- Lightning arcs now last longer, fade out, and render with a softer glow + jaggier bolt body to reduce harsh flashiness.
